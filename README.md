# Game Shop 🎮

A simple gaming website project designed to display and explore games in a clean and modern way.

## Pages

### 🏠 Home Page
The main page of the website that introduces the platform and highlights featured content.

### 🔥 Top Games
A page that showcases the most popular and trending games.

### 🎮 All Games
Contains a collection of all available games with their basic information and images.

### 👤 Profile Page
A simple user profile page displaying user information and activity.

---

## Features
- Modern gaming-style UI
- Simple and responsive design
- Easy navigation between pages
- Organized game sections

## Technologies Used
- HTML
- CSS
